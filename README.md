2. Aim of the Project

The aim of this project is to create an interactive, user-friendly web portal that allows students to test their knowledge and learn different subjects through quizzes and flashcards. The system helps learners improve their skills in Aptitude, HTML, JavaScript, and General Knowledge using a simple and effective interface.

6. Working Process (How the Website Works)

1. The user opens the **Home Screen**, where they select a quiz category.
2. They choose between **Start Quiz** or **Flashcards**.
3. In **Quiz Mode**, questions load one by one with answer options.
4. When completed, the user sees their total score on the **Result Screen**.
5. In **Flashcard Mode**, users flip cards to view answers and navigate through all cards.
6. At any time, users can return to the **Home** screen.
